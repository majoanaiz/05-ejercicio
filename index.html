Sobre el Proyecto

Este es un ejemplo de cómo estructurar una página web de una sola vista para garantizar que sea completamente adaptable. Utilizamos clases de Tailwind como md:flex y max-w-4xl para controlar el diseño en diferentes tamaños de pantalla, evitando el desbordamiento o que los elementos se vean "aplastados" en el móvil.
